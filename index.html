Lightweight Learner for Shared Knowledge Lifelong Learning

Yunhao Ge1, Yuecheng Li1,
1Thomas Lord Department of Computer Science, University of Southern California, 2Neuroscience Graduate Program, University of Southern California
Paper arXiv Code Data

LLL enables cheap communication between numerous learning agent in different areas with their own unique knowledge.

Abstract

We present an innovative approach to a unique novel challenge where multiple agents, situated in diverse locations, need to communicate their learned knowledge with each other. Our method ensures minimal cost while facilitating effective communication.

  • Shared Knowledge

    Our approach allows multiple agents to independently learn and acquire unique knowledge. They can then swiftly share this knowledge with each other, enhancing the collective intelligence of the system.


  • Lightweight

    Our method is designed to be efficient. The time and space required for communication between agents are minimal, making the process highly efficient.


  • Lifelong Learning

    Our system is designed to retain old knowledge even after new knowledge is acquired from other agents. This feature prevents catastrophic forgetting, ensuring a continuous learning process.

Key Idea

Each agent in our system possesses a pre-trained general model with a small, flexible component tailored to one or more specific tasks. Simultaneously, each agent has a task mapper that helps identify the task at hand. The flexible component can be shared and transferred among agents, with the task mapper deciding which component to use.

Method

Our primary methodology focuses on image classification tasks. We utilize weights pre-trained on ImageNet as our fixed backbone. After each convolutional layer, we insert a learnable bias layer that varies for each task. Additionally, each agent learns a task-specific linear head for classification. For the task mapper, we extract features from all images using a fixed pre-trained ResNet-50 backbone, and a Gaussian mixture model is learned for these features to facilitate future task classification.
